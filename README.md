## • Overview

This plugin exposes vscode like api in Acode.

## • Installation

1. Open the plugins sidebar on Acode
1. Search for **vscode-api**
1. Click Install

## • Usage

```js
const vscode = acode.require("vscode");
```

## • Note

Not all vscode's api is/will be implemented.
